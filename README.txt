# Ejemplo de Uso de Azure Cognitive Services para Análisis de Sentimientos

Este repositorio contiene un ejemplo simple de cómo utilizar Azure Cognitive Services para analizar el sentimiento de un conjunto de documentos de texto usando Python.

## Requisitos Previos

Asegúrate de tener los siguientes requisitos previos antes de ejecutar este código:

1. Una suscripción activa de Azure.
2. Una clave de API válida para Azure Cognitive Services.
3. Python instalado en tu sistema.

## Instalación

Puedes instalar las dependencias necesarias para este proyecto utilizando pip:

```bash
pip install azure-ai-textanalytics
